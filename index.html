Core Features
What It Does
01
[ 01 ] Enforcement
Rate Limiting Engine
Token bucket, sliding window, and fixed window algorithms. Configurable per key, per endpoint, or per IP.
[ 02 ] Management
API Key Control
Generate, revoke, and scope API keys in seconds. Attach custom rate tiers and expiration policies.
[ 03 ] Visibility
Live Analytics
Real-time request counters, throttle events, and usage heatmaps. Know exactly who's calling what, and when.
[ 04 ] Backend
Spring Boot Native
Drop-in Java filter with zero config overhead. Plug into any Spring Boot REST controller in minutes.
[ 05 ] Scale
Redis Backed
Distributed counters via Redis for horizontal scaling. No state drift across nodes, no false positives.
[ 06 ] Response
Smart 429 Handling
Structured error payloads with retry-after headers. Clients always know when to back off and try again.
